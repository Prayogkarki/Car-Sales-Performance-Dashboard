# 🚗 Car Sales Performance Dashboard | Power BI

## 📌 Overview
This project is a **Power BI dashboard** designed to analyze car dealership sales performance across regions, car models, and time periods.

The dashboard provides interactive insights into key metrics such as **YTD, MTD, and YoY sales**, helping stakeholders make data-driven decisions.

---

## 🎯 Objective
- Track sales performance across regions and time  
- Analyze trends in sales, revenue, and cars sold  
- Identify high-performing and low-performing segments  
- Compare current year vs previous year performance  

---

## 🛠 Tools Used
- Power BI  
- DAX (Data Analysis Expressions)  
- Data Modeling  

---

## 📊 Key KPIs
- Year-to-Date (YTD) Sales  
- Month-to-Date (MTD) Sales  
- Year-over-Year (YoY) Growth  
- Total Cars Sold  
- Average Price  

---

## 📈 Dashboard Features

- Interactive filters (Year, Region, Car Model)  
- KPI cards for key metrics  
- Sales trend analysis (weekly/monthly)  
- Regional performance visualization (map/bar charts)  
- Sales breakdown by car model and category  
- Detailed data table for deeper insights  

---

## 🔍 Key Insights
- Identified top-performing regions contributing highest revenue  
- Analyzed best-selling car models  
- Detected sales trends over time  
- Compared performance with previous year
